# C問題

1122文字列を構成する部分文字列の数を数える．このとき，ランレングス圧縮を用いて，$O(N)$で計算する工夫をする．

（例）11122311

この文字列には，12, 1122, 23が含まれるため，答えは3となる．

この文字列をランレングス圧縮すると，$(1, 3)(2, 2)(3, 1)(1, 2)$となる．各要素は，対象の数値とその連続個数を表している．
この次の数値が1だけ大きいとき，その隣り合う連続個数の小さい方の数値を足していくことを繰り返して，解答を得る．
$(1, 3)(2, 2)$からは，少なくとも1122の部分があることが言えるから，この内には2つの1122文字列を含むことを示しているということである．
